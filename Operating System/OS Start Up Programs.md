想要在操作系统启动时自动运行一些服务！ (没错就是 [frp](https://github.com/fatedier/frp))

## Windows

将自己写好的 `.bat` 脚本放入 `C:\ProgramData\Microsoft\Windows\Start Menu\Programs\StartUp` 中。

## Linux

将自己写好的 `.sh` 脚本放入 `/etc/profile.d` 中。(执行环境变量以外的程序需要写绝地路径)
