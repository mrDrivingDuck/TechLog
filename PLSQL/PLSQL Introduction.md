## About

__PL/SQL__ (Producedural Language/SQL) 是一种程序语言

是 _Oracle_ 数据库对 SQL 的扩展

在普通 SQL 的语句上增加了编程语言的特点

---

## The Generation of Programming Languages

* 1GL - 机器语言 (打孔机)
* 2GL - 汇编语言
* 3GL - 高级语言
* 4GL - __非过程化__ 程序设计语言
    * 对某些标准处理过程进行自动生成
    * 用户只说明要做什么，具体的执行步骤安排给软件自动处理
    * 如 SQL

PL/SQL 属于 3GL
